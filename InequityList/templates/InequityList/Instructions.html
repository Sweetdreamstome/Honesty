Instrucciones

En esta parte del experimento se le presentará una lista con 21 asignaciones de dinero para usted y otra persona. Para cada asignación usted podrá indicar qué tan satisfecho se encuentra con esa asignación. Para esto al lado de cada asignación hay una barra que la puedes mover para indicar su nivel de satisfacción, la cual va desde 1 hasta el 7, donde 1 representa que estás "Extremadamente Insatisfecho" y 7 "Extremadamente Satisfecho".

Recuerde que para esta parte del experimento cada Punto equivale a 0.05 Soles.
